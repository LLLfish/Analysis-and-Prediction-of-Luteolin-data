# 木犀草素数据分析与预测系统

## 项目介绍
本系统是一个对木犀草素(LUXR)的分析与预测系统，从从PubChem、ChEMBL、TCMSP等数据库中获取蛋白质特征，基于深度学习和图神经网络的多维度分析，用于预测木犀草素可能的靶点

## 技术栈
现有技术栈组合如下：
```
┌─────────────────────────────────────────────────────────────────────────────────┐
│                         技术栈组合                                           │
├─────────────────────────────────────────────────────────────────────────────────┤
│                                                                                 │
│  数据层:     Pandas + BioPython + PubChemPy                                    │
│                 ↓                                                               │
│  GNN模块:    PyTorch Geometric (GAT/GCN)                                       │
│                 ↓                                                               │
│  CNN模块:    DeepChem (3D-CNN) + SchNetPack                                    │
│                 ↓                                                               │
│  蛋白模型:   ESM-2 (HuggingFace) + ESMFold                                     │
│                 ↓                                                               │
│  分子处理:   RDKit (替代 Open Babel)                                           │
│                 ↓                                                               │
│  特征融合:   Transformers (Multi-Head Attention)                               │
│                 ↓                                                               │
│  分子对接:   GNINA (CNN评分) + AutoDock Vina                                   │
│                 ↓                                                               │
│  知识图谱:   PyKEEN + DGL-KE                                                   │
│                 ↓                                                               │
│  可视化:     Web                                                               │
│                                                                                 │
└─────────────────────────────────────────────────────────────────────────────────┘
```
